# Gitea Mirror Runner

This repository is used to mirror all my Repository / Gists from github to my selfhosted gitea instance. 

## Things It Mirrors
1. Repository Created
2. Repository Stared
3. Repository Forked
4. Gists Created
5. Gists Stared
